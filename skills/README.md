# skills/

Reusable patterns extracted from real work, in `SKILL.md` format. This is where
the **ratchet effect** lives: when `/loop` (or you) hits a failure worth
remembering, encode the lesson here so it never recurs.

## Format

Each skill is one directory with a `SKILL.md`:

```
skills/<skill-name>/SKILL.md
```

Frontmatter: `name`, `description` (used for recall), optional `domain`.

> 🚧 Empty by design — fills up as you work and ratchet lessons in.
